# Dog Hostel Website (React JS)

A modern and responsive Dog Hostel website built using React JS.  
This project provides a clean UI for showcasing dog hostel services, gallery, and basic layout components like header and footer.

---

## 🚀 Features

- 🐶 Modern pet-friendly UI design
- 🖼️ Image gallery with hover effects
- 🎨 Dark modern theme styling
- 📱 Fully responsive design (mobile + desktop)
- 🧩 Reusable components (Header & Footer)
- ⚡ Fast and lightweight React application

---

## 🛠️ Tech Stack

- React JS ⚛️
- HTML5
- CSS3
- JavaScript (ES6+)
